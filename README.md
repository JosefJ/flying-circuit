# Flying Circuit

Website of Hackatoshi's Flying Circuit

## Development

```bash
git clone git@github.com:ParalelniPolis/flying-circuit.git
cd flying-circuit
npm install # or yarn
npm run dev
```

## Deploy

WIP!

```bash
npm run export
git subtree push --prefix out origin gh-pages
```
